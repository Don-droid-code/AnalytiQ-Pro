# AnalytiQ Pro

**AI‑powered business valuation platform – transparent, educational, open source.**

AnalytiQ Pro merges **data cleaning** and **financial analysis** into a single tool.  
It helps SMEs, analysts, and students evaluate a business across **6 dimensions**:

- Financial
- Market
- Operations
- Intangibles
- Human Capital
- Risk / ESG

Every transformation generates **visible, editable SQL/Python code**.  
No black boxes – you see how the result is built.

---

## ⚙️ Tech Stack (MVP – V1.4)

- **Orchestrator:** LangGraph (Open‑source)
- **Intelligence:** Gemini 1.5 Pro / Flash (Google AI Studio)
- **Database & ORM:** SQLite + SQLAlchemy
- **Data Validation:** Pydantic
- **Backend:** FastAPI (Python)
- **Frontend:** React / Vue.js
